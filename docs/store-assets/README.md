# Chrome ウェブストア画像アセット

Chrome ウェブストア掲載用の画像一式。サイズ要件は
[`../store-listing.md`](../store-listing.md) を参照。

## スクリーンショット（1280x800・最大5枚）

| ファイル | 内容 |
|---------|------|
| `01-hover-highlight.png` | 要素ホバーでハイライト（拡張を起動した状態） |
| `02-pin-popover.png` | 要素クリック→ピン配置＋ポップオーバーで指示入力・要素サムネ |
| `03-inbox-list.png` | Inbox に複数アノテーションが一覧（タグ・指示・サムネ） |

いずれも実際に拡張機能を架空の SaaS ダッシュボード（React 製。`__next_f` を
持たせて Next.js (App Router) として検出させたデモアプリ）上で操作して撮影した
実画面。

- 02 / 03 の要素サムネイルは、自動操作下では `chrome.tabs.captureVisibleTab`
  が空フレームを返すため、Playwright の要素スクリーンショットを注入して
  **実機での見え方を再現**したもの（写っている要素自体は本物）。
